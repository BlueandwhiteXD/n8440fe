\subsection{天使大人的圣诞约定}

% 「クリスマスの予定ですか？　特にありませんけど」\\
「圣诞节的安排吗？　应该没有吧」\\

% 洗い物が終わった後にソファに腰かけていた真昼に問いかければ、実にあっさりとした返答があった。\\
周洗完碗，向坐在沙发上的真昼问到，得到了十分干脆的回答。\\

% てっきり女子会をしたりするのかと思っていたのだが、予定はなかったらしい。
本以为八成会有个女生聚会什么的，但好像真昼并没有这种预定。

% 意外だと顔に出ていたらしく、真昼は周を見てほんのりと呆れた顔を浮かべている。\\
或许是看见了周那一脸意外的表情，看向周的真昼脸上露出了一丝不解。\\

% 「基本的に、私と交友のある同性の級友は大体彼氏が居ますよ。男性から誘われてもお断りしていますから、予定はどうしても空きますね」
「基本上，那些和我有交情的女同学大多是有男朋友的。男性的话就算来邀请我也会拒绝，所以不管怎么说都是没有安排的」

% 「男子涙目だな」\\
「男的听了都要哭了」\\

% 外行きの際の真昼は非常にガードが固く、淡い期待を込めて真昼を誘った男子達は堅牢な守備に涙を飲んだ事だろう。
外出时的真昼防御极为坚固，那些怀着淡淡的期待邀请真昼的男生们在这坚固的防御面前也只得默默流泪吧。

% 周からしてみたらよく誘えたな、という感想である。自分に自信がなければあの天使様を誘えないだろうし、陽キャはすごいなと感心すらしていた。\\
周是觉得，真亏那帮人干的出邀请真昼这事啊。要是对自己没有足够的自信，邀请天使大人简直是想都不敢想，周反而佩服起那些性格阳光的家伙们来了。\\

% 「……そんなに私と過ごしたいものなのですかね」
「……和我一起过圣诞的吸引力居然这么大呢」

% 「あわよくばお近づきになりたいんだろ」
「幸运的话可以拉近距离嘛」

% 「なんのために？」
「为了什么？」

% 「そりゃ、お付き合いしたいからでは？」
「要说的话，想要交往？」

% 「何故付き合いたいのでしょうか」
「为什么想要交往呢」

% 「……付き合ってあれこれしたいんじゃないのかね」
「……要论交往还不是想做些这个那个的嘛」

% 「不純ですね」\\
「动机不纯呢」\\

% ばっさり切り捨てられている男子諸君に内心合掌しつつ「まあでも」と付け足す。\\
周在心里对被干脆地无视了的诸位男生合掌致意，「嘛，不过」补了一句。\\

% 「そういうやつばっかりじゃないと思うから、あんま疑ってやるな。お前なら、男が向けてくる視線の質くらい分かるだろ」
「应该也不全是那样的家伙啦，别太疑神疑鬼啊。你的话应该能明白男生看向你的视线的类型吧」

% 「そうですね。全員が全員不埒な眼差しで見てくる訳でもありませんし。周くんだって違うでしょう？」
「也是呢。要论全部的话倒也并不全都是无礼的目光。比如周你的就不是对吧？」

% 「俺がお前を不埒な目でいつ見たんだ」\\
「那我倒真没干过呐」\\

% 可愛いとか頭撫で回したいとかその程度の考えならよぎった事はあるが、どうこうしたいとまでは考えていない。
可爱啊想摸摸头啊这种程度的事倒是经常浮现在脑子里，不过不过要论想入非非的程度那倒没有。

% そもそもそんな事を考えていたら真昼が気付いて離れていただろう。\\
再说要是想了那种事情，真昼就会查觉到然后疏远自己吧。\\

% 無害な男だから彼女の隣に居座れるのであって、仮に牙の一つでも見せたら彼女はすぐに居なくなる。\\
因为是无害的男人所以才得以坐在她身边，要是哪怕露出了一点凶相，她就会立刻离开自己吧。\\

% 特にこれといって彼女ほしいという願望はないし食欲の方が大切なので、今の関係を崩すつもりもなかった。\\
要论的话周倒也并不怎么想要女朋友，不如说反而是食欲这边比较重要，因而周暂且也没有让现在的关系崩溃的想法。\\

% 「そうでしょうね。周くん最初から私に興味なさそうでしたし」
「也是呢。周君的话看起来一开始就对我没什么兴趣」

% 「まあ」
「嘛」

% 「ですので、信頼してます」
「所以说，可以信任」

% 「そりゃどうも」\\
「那可是感激不尽」\\

% 男としてそれでいいのかという信頼のされ方をされている気がしなくもないが、ひとまず安全な男の立ち位置に不満はない。\\
虽然这信赖的方式感觉对男人来说有些不太好接受，不过对「安全的男性」这一立场周暂时没有什么不满。\\

% 「……で、私のクリスマスの予定を聞いてきた周くんは何か予定が？」
「……于是乎，问完了我安排的周君的安排如何？」

% 「ん？　ああ、俺、二十四日は昼に樹達がここにくるから、まあいつも通りだが晩ご飯が遅くなるかもしれんっていう連絡しようと思ってな」\\
「嗯？啊啊，我的话二十四号白天树他们会过来我这，虽然预定不变，但晚饭可能会比平常要晚一些，所以想着先跟你提前说下」\\

% ようやく話が本筋に戻ってきたので改めて説明する、真昼は納得したように頷いた。\\
总算是又绕回了正题，周再次解释了下，而真昼则是明白了一般点了点头。\\

% 「分かりました。そのクリスマスパーティーが終わったら呼んでください、そこからご飯作りますので。用意だけはしておきますね」
「我知道了。那就等圣诞 party 结束之后叫我，我再过来做饭。之前的准备就要你来咯」

% 「おう、なんかすまん」
「哦，嘛抱歉了」

% 「いえ。楽しんでください」
「没事，祝你过得开心」

% 「……寂しくないのか？」
「……不觉得寂寞吗？」

% 「慣れてますし、一人は」\\
「一个人已经习惯了，毕竟」\\

% 何て事のないように告げられて、少しだけ胸が痛んだ。
真昼一副没事的样子回答道，却让周心里很不好受。

% 彼女の脳内でも親の事が一瞬よぎったのかもしれない、どこか自嘲するような笑みが浮かんでいる。\\
而或许真昼的脑内也突然闪过了父母的事情，使她的脸上浮现出略带几分自嘲的苦笑。\\

% 「……あー、その」
「……啊——，那个」

% 「はい？」
「怎么了？」

% 「……非常におこがましい申し出ですが、イブは無理でもクリスマス一緒に居る、というのは」\\
「……虽然是个十分冒昧的请求，就算圣诞前夜不行，圣诞节那天可不可以一起，这样」\\

% なんというか、こういった提案をするのは、非常に気恥ずかしい。
不知为何，提出这样一个提议让周十分的害羞。

% 特に他意はないのだが、クリスマスを共に過ごそうと誘うのは、普通特別な意味を持つのだ。\\
虽然周并没有什么特别的意思，但一般来说，一起过圣诞节的请求，往往是有着特别的意义的。\\

% 決して、他意はない。
周真的，没有别的意思。

% ただ、真昼がどこか寂しそうな眼差しで瞳を伏せたのが、嫌だっただけなのだ。\\
只不过是，不愿看见真昼那一副寂寞的表情低着头的样子罢了。

% 提案に、真昼は目をぱちりと瞬かせていた。\\
听见这一提议，真昼不觉地眨了眨眼。\\

% 「一緒にって、何するのですか？」
「在一起，是要做什么吗？」

% 「え？　あー、別にする事ないよな。ごめん」\\
「诶？啊——，其实并不是要做啥。抱歉」\\

% そこを指摘されると、周としてはそれ以上は強く押せない。\\
被指出那一点，周就没有了有说服力的理由了。\\

% 他人に見かけられた場合の面倒を考えれば一緒に出かける事はまずない。
要是考虑被他人看见的情况造成的麻烦，首先一起出门的选项就被否决掉了。

% では家で過ごすという事になるが、この家には真昼の興味を引くようなものはほとんどないだろう。\\
那就只剩在待在家里了，但这个家里也几乎没有什么可以引起真昼兴趣的东西。\\

% となると何もせずに二人で側に居るだけなのだが、恐らくめちゃくちゃ気まずいのではないだろうか。\\
这么一来只剩下两人在一起什么都不做这一个选项，但那样一来气氛怕是会变得十分尴尬。\\

% それくらいなら、別々に過ごした方がいいのかもしれない――そういう考えで撤回しようとしたのだが、真昼は周を静かに見つめていた。\\
要是这样的话，还不如两人各过各的还更好——周如此考虑着，正想着撤回自己刚刚的提议，真昼却静静地看着周。\\

% 「……じゃあ、あれ、してみたいです」\\
「……那、我想试试，那个」\\

% 予想外だったのは、真昼が存外乗り気だった事だろうか。\\
真昼对此抱有兴趣，令周颇感意外。\\

% か細い指が、テレビの方を示す。
真昼纤细的手指，指向了电视的方向。

% 正しくいうと、テレビボードの中に格納されているゲーム機だろう。
准确来说，是电视柜里放着的游戏机吧。

% 最近の夕方は真昼が居るのであまり起動しなくなったのだが、それに興味を示したらしい真昼は「ああいうの、した事ないですし……」と小さく希望を示している。\\
最近晚上的时候真昼经常在家里，因而周也没怎么碰过，可似是对它感兴趣的真昼却「那个东西，我还没有玩过……」地小声表达着自己的期望。\\

% クリスマスに特に交際している訳でもない男女がゲームして過ごすというのは何だかシュールな気もする。\\
一对并没有什么特别关系的男女玩着游戏过圣诞节——听上去甚至有种超现实的感觉。\\

% 「いや、まあ、それでもいいけど……いいのか？　ゲームとかで」
「呃，嘛，也不是不行……可以吗？玩游戏什么的」

% 「駄目なのですか？」
「不行吗？」

% 「駄目ではないが」
「也不是说不行啦」

% 「じゃあ、それでいいです」
「那，这样就好」

% 「お、おう」\\
「呃，哦」\\

% それでいいんだ……とは思ったものの、真昼の希望なので出来うる限り叶えてやろうとも思う。
就这样就好么——虽然有点担心，但既然这是真昼的希望，周还是决定尽自己所能将其实现。

% ささやかな楽しみくらい、あげたい。特にクリスマスに予定を入れている訳でもないし、真昼と食事が出来るだけでももうけものだろう。\\
周想要给真昼，一些小小的乐趣。不过反正周圣诞节也没有什么特别的安排，对周来说就算只是能和真昼一起吃饭也算是赚到了吧。\\

% 「ま、クリスマスとか関係なしにまったり過ごせばいいだろ」
「嘛，也不管什么圣诞节了，悠闲地过就好了咯」

% 「そうですね」\\
「是呢」\\

% 小さく笑った真昼が何だか直視しにくくて、周はうなずいてさりげなく顔を背けた。
轻轻一笑的真昼不知为何，让周感到难以直视，周一边点着头一边漫不经心地别开了脸。
